What are the different ways to select an element in the DOM?
  • Using getElementById.
  • Using getElementsByClassName.
  • Using getElementsByTagName.
  • Using querySelector.
  • Using querySelectorAll.
What is the difference between innerHTML, innerText, and textContent ?
  • textContents is all text contained by an element and all its children that are for formatting purposes only.
  • innerText returns all text contained by an element and all its child elements.
  • innerHtml returns all text, including html tags, that is contained by an element.
What is event delegation in the DOM?
In the DOM (Document Object Model), "event delegation" is a technique where you attach a single event listener to a parent element, allowing it to handle events triggered on its child elements, instead of adding separate listeners to each child element
What is event bubbling in the DOM?
Event bubbling is when an event on a child element propagates up to its parent and ancestors. It can be stopped using event.stopPropagation().
How do you create, add, and remove elements using JavaScript?
  1. Create: let el = document.createElement("div");
  2. Add: document.body.appendChild(el);
  3. Remove: el.remove(); or parent.removeChild(el);
